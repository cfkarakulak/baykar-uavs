### Installation

Commands to run

```
git clone git@github.com:cfkarakulak/baykar-drf.git
cd baykar-drf
docker-compose up
```

Once built,

Navigate to http://localhost:5173/login

You can either use `baykar:bayraktar` to log in, or register a new account on /register
